Lambda is a demo web application. It is written in python using the Django library. On the front end it makes heavy use of Bootstrap for it's styling and custom Javascript to add functionality. The management system is based around three core concepts; teams, projects and tasks, read more about these below.

Source 

Teams

With Lambda you work as part of a team, this can include only yourself or your entire organisation.

Projects

Teams are assigned to projects which can be anything from you daily todos to a product you are going to launch.

Tasks

Projects get assigned tasks, each one representing a step towards completing the project.
